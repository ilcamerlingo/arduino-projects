# uStepper S32 - Adresses 100-101

## Description
Contrôle de moteurs pas à pas avec uStepper S32 pour modélisme ferroviaire.
Gestion des adresses 100 et 101 via protocole DCC.

## Matériel
- uStepper S32 board
- Moteur pas à pas NEMA 17
- Alimentation DCC

## Bibliothèques
- uStepper S32 library
- STM32 Core pour Arduino

## Procédure de programmation
1. Brancher uStepper S32 au PC
2. Presser et maintenir "BOOT"
3. Presser "RESET", relâcher, puis relâcher "BOOT"
4. Cliquer "Upload" dans Arduino IDE
