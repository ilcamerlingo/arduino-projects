# DCC Basic Accessory Decoder

## Description
Décodeur DCC pour accessoires de modélisme ferroviaire.

## Matériel
- Arduino compatible DCC
- Librairie DCC_Decoder

## Fichiers
- DCC_Basic_Acc_Decoder.ino : Sketch principal
- DCC_Decoder.h : Header de la librairie
- DCC_Decoder.cpp : Implémentation de la librairie
